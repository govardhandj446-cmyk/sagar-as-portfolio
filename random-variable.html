Random Variables

A random variable maps outcomes from an experiment to numerical values. It can be discrete or continuous.

Types of Random Variables

Discrete

Countable values such as the number of heads in coin tosses or number of students present.

Continuous

Measured values that can take any value in a range, such as height, weight, and time.

Common Distributions

  • Binomial describes the number of successes in fixed trials with constant probability.
  • Poisson models rare events in a fixed interval, such as arrival counts or defects.
  • Normal is the classic bell curve used for measured data like exam scores.
  • Exponential describes time between events, including waiting times or failure intervals.

Learning Outcomes

  • Define discrete and continuous random variables.
  • Describe probability mass and density functions.
  • Build distribution tables and compute probabilities.
  • Use binomial, Poisson, normal, and exponential models.
  • Apply distributions to real-world situations.
