Understanding Relationships

Correlation and regression help analyze how two variables move together and how one variable can predict another.

Correlation

Positive Correlation

Both variables increase together.

Negative Correlation

One variable increases while the other decreases.

No Correlation

No visible linear relationship between variables.

Pearson & Spearman

  • Pearson’s r measures linear correlation between two continuous variables.
  • Spearman’s rank measures monotonic relationships when data are ranked.
  • Probable Error checks whether the computed correlation is meaningful.

Regression

Regression finds the best-fitting line to predict one variable from another, using equations like Y on X and X on Y.

Learning Outcomes

  • Interpret strength and direction of relationships.
  • Compute and apply correlation coefficients.
  • Construct regression equations for prediction.
  • Understand the difference between correlation and causation.
  • Use regression for forecasting and data analysis.
