Selected Python examples from lab assignments demonstrate how statistical concepts are implemented with code.

Example 1: Summary Statistics

import numpy as np
import matplotlib.pyplot as plt

data = [75,89,66,52,90,68,83,94,77,60]
mean = np.mean(data)
median = np.median(data)
std_dev = np.std(data)
print("Mean:", mean)
print("Median:", median)
print("Standard Deviation:", std_dev)
plt.hist(data, bins=5)
plt.title("Histogram")
plt.show()

Example 2: Probability with Cards

total_cards = 52
queens = 4
hearts = 13
queen_of_heart = 1
f_outcome = queens + hearts - queen_of_heart
probability = f_outcome / total_cards
print("Probability of queen or heart:", probability)
